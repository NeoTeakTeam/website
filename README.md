# NeoTeak Team Website

这里是 NeoTeak Team 的网页源代码

该项目随后会部署到 <neoteak.cn> 中.
 而 blog 将会搬迁到 <blog.neoteak.cn>

## 使用

首先, 你要确保你的系统有以下环境

|环境|版本|介绍|
|---|---|---|
|Node.js|Any|Vue运行所需|
|npm/pnpm|Any|安装环境|

1. 克隆仓库

```bash
git clone https://github.com/NeoTeakTeam/website.git
```

2. 进入并安装环境

```bash
cd website
npm install  # Use npm
pnpm install # Use pnpm
```

3. 运行开发环境

```bash
npm run dev  # Use npm
pnpm run dev # Use pnpm
```
